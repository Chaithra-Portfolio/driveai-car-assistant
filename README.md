# 🚗 DriveAI Car Assistant

AI-powered car dealership website with dynamic navigation and chatbot interaction.

## 🌐 Live Demo
https://driveai-car-assistant.netlify.app

## 💻 GitHub Repository
https://github.com/Chaithra-Portfolio/driveai-car-assistant

## ⚙️ Tech Stack
- React.js
- JavaScript
- HTML/CSS
- Netlify (Deployment)
- GitHub (Version Control)

## ✨ Features
- AI chatbot navigation
- Car filtering and comparison
- Dynamic UI updates
- Booking form with pre-filled data
- Smooth scrolling to sections

## 🤖 Supported Queries
- "Show me SUVs under 20 lakhs"
- "Compare top models"
- "Book a test drive"
- "Best car for family"
- "Show prices in dollars"
- "Show budget cars"

## 🛠️ Setup Instructions
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
